1. Download ZIP
2. Extract folder
3. Open command line
4. Make sure you have latest .NET SDK installed on your computer
5. Navigate to extracted folder
6. Type command: dotnet run --project RandomProject1
7. Choose continent
8. Choose how many countries you want to display
9. Press show button
